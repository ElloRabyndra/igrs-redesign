# IGRS Redesign

This project covers the redesign of the IGRS (Indonesian Game Rating System) website [igrs.id](https://igrs.id/), focusing on three key aspects: UI/UX, Frontend, and Backend.

The **UI/UX** redesign aims to deliver a more modern, intuitive, and user-friendly interface that better communicates game rating information to the public. The **Frontend** will be rebuilt to reflect the new design system with cleaner structure, better performance, and improved maintainability. The **Backend** will be restructured with a well-defined database schema and RESTful API endpoints to support the new frontend needs.
